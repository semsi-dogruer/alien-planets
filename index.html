Terminal and Git

You are going to build this assignment only using the terminal (no clicking in the system file explorer or VS code to create, copy, delete, or move files).

  1. In VS Code settings, search for Import Module Specifier Ending and set it to .js/.ts
  2. Create a directory to store the files for this assignment and call it alien-planets (remember: you can only use the terminal).
  3. Create a file called index.html.
  4. Make your first git commit and push to a new repo on GitHub.
  5. Create an index.js file. (You aren't using VS Code, ARE YOU?)
  6. Create a script tag with an attribute of type="module" and link it to index.js as normal.
  7. Create an alienPlanets.js file.
  8. Create a style.css file.
  9. Copy and paste the array from Canvas into alienPlanets.js (no terminal required for this step).
  10. Build the rest of this website without moving the array from alienPlanets.js
